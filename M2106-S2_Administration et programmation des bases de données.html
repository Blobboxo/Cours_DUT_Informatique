[MS ACCESS] Étapes de création de formulaire et connexion événementielle (boutons) à l'aide des librairies DAO et ADO :


Étape 1 : "Placer un formulaire vide". Onglet « Créer » / « Création de formulaire »


Étape 2 : "Modifier les options de création". Aller dans la feuille de propriétés du formulaire créé et choisir les options suivantes dans l’onglet « Format » :
• Style bordure : Aucun
• Afficher sélecteur : Non
• Barre défilement : Aucune


Étape 3 : "Créer les boutons du formulaire". [Mode création sur le formulaire] Onglet « Création » / Créer un « Bouton » et le placer à l’endroit voulu / Cliquer sur « Annuler ».

Remarque : Pour donner un nom au label du bouton, il faut double-cliquer dessus en mode création pour le formulaire (ex : Lecture). Pour modifier la légende d’un bouton, il faut aller dans l’onglet « Autres » puis modifier le champs « Nom » dans la « Feuille de propriétés » du bouton correspondant.


Étape 4 : "Création code VBA pour chaque procédure événementielle assignée à un bouton". [Mode création sur le formulaire] Cliquer sur un bouton / Cliquer sur « Événements » dans la « Feuille de Propriétés » / Cliquer sur « ... » à droite du type d’évènement voulu (clic, double-clic…) et choisir dans la fenêtre « Générateur de code l’option : « Générateur de code ». Une fois que la nouvelle fenêtre s’ouvre, saisir le code correspondant.

Remarque : À chaque fois qu’une procédure événementielle (macro) est créée ou modifiée, il faut sauvegarder son nom pour l’exécuter.
