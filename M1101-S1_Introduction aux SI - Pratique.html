Commandes basiques :

Commande Signification
pwd Print Working Directory = Afficher chemin répertoire de travail (=répertoire courant, celui où l’on se trouve)
[commmande] --help Affiche l’aide de la commande correspondante (ex : pwd –help) (moins détaillé que man [commande])
man [commmande] Affiche la page du manuel correspondant à la commande spécifiée (appuyer sur q pour quitter)
ls [repertoire] [option] Liste le contenu d’un répertoire, y compris les fichiers ET dossiers cachés
ls .. Liste le contenu du dossier PARENT
ls -l Liste le contenu d’un répertoire avec les droits d’accès (d = dossier, sinon c’est un fichier)
ls -la Liste le contenu d’un répertoire avec les droits d’accès + fichiers cachés
cd [repertoire] Change Directory (changer de répertoire de travail = se déplacer dans l’arborescence)
cd .. Remonte d’un cran dans l’arborescence, dans répertoire PARENT
mkdir [dossier1] [dossier2] Crée un ou plusieurs dossiers (en spécifiant leurs noms) là où on se trouve
mkdir [nom\ dossier] Permet de créer UN dossier possédant un nom composé d'un espace au milieu (le "\" est un caractère d'échappement de l'espace qui suit)
touch [fichier1] [fichier2] Crée un ou plusieurs dossiers (en spécifiant leurs noms) là où on se trouve
touch [destinat.] / [fichier] Crée un fichier à l’emplacement du dossier spécifié (peut faire même chose avec un dossier)
mv [fich./doss.] [destinat.] Déplacer un fichier/dossier d’un emplacement à un autre
mv [fich./doss.] [nveau nom] Renommer un fichier/dossier (et écraser le précèdent)
cp (-r) [fich./doss.] [destinat.] Copier un fichier/dossier vers un autre emplacement. Pour un dossier, ajouter l’option “-r” devant “cp”
rm [fichier] Supprimer un fichier
rm -r [dossier] Supprimer un dossier (/!\ Supprimer un dossier, qui est un PARENT supprime tous les ENFANTS !!!)
[commande] > [destination] Redirige la sortie d’une commande et l’envoie ailleurs (ex: ls > […] redirige ce qui est affiché grâce à “ls”).
cat Afficher le contenu d’un fichier
more [fichier] Afficher de façon lisible le contenu d’un fichier (navigation au clavier) qui est très long (q pour quitter)
less [fichier] Même chose que less avec plus d’options (q pour quitter)
grep [mot] [nomfichier] Afficher le contenu d’un fichier sans l’ouvrir (ex: affichera la phrase correspondante à un mot saisi )
* Caractère “joker” : Peut être utilisé comme substitut pour n’importe quel caractère dans une recherche)
cd ./[dossier] Accès au dossier depuis le répertoire COURANT
cd ~ Accès au répertoire ACCUEIL utilisateur ($HOME) → “cd” fait la même chose
$* Liste des arguments
$# Nombre de paramètres passés au script
$1, $2, $2… Nombre du script passé (par ordre de position lors de l’exécution)
$0 Nom du script exécuté
N.B. : [mot] Les mots entre crochets sont des paramètres dont la valeur dépend du contexte considéré (ex : le nom du dossier peut s'appeler "toto" ou "titi").
